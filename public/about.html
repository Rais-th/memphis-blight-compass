About the Blight Compass

A single public interface for the datasets Memphis already publishes but rarely joins. Built for the Blight Strike Team, resident advocates, journalists, and anyone who has tried to report the same abandoned lot three times and wondered where the request went.

Scoring formula

score = (chronic_complaints × 3)     chronic = 3+ 311 requests in last 12 months
      + (code_violations × 2)         count of Code Enforcement cases
      + (1 if flood_zone is X)        FEMA Zone X or SFHA=False
      + (1 if asking_price < $2000)
      + (1 if width ≥ 30 AND length ≥ 50)

Higher scores = more neighborhood pain concentrated on a single parcel, with attributes that make intervention practical. This ranks parcels, it does not judge owners. Scores are recomputed each ingestion run.

What this tool does not do

  • Does not accept applications or transfer ownership.
  • Does not publish owner-level editorial claims. Owner fields are hidden; apply via the Land Bank portal for official records.
  • Does not include eviction, tenant, or non-public court data.
  • Does not constitute legal, financial, or real-estate advice.
