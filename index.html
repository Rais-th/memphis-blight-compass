Why this exists

The Bloomberg Harvard City Leadership Initiative publicly documented in October 2023 that Memphis's 311 integration "has not yet been solved" — complaints don't route effectively between Code Enforcement, Public Works, and Solid Waste. Director Melanie Neal of Community Enhancement said in October 2025: "We need a faster way to intervene."

Meanwhile, Shelby County Land Bank inventory isn't cross-referenced with 311 blight data. Parcels sell for $400 to $2,000 in zones where neighbors have complained for years. Nothing connects the data — until now.

How to use this

Blight strike team

Open the map and sort by score. Each red dot is a parcel with 3+ complaints in 12 months or multiple active code cases. The equity dashboard shows which zips wait longest for resolution.

Residents and non-profits

The top 50 list surfaces acquisition-ready lots where the City-owned Land Bank has them for sale under $2,000. Click any parcel for full complaint + violation history.

Journalists and researchers

All data is public record. CSV exports and an open API power this page. See methodology for sources and caveats.
